##### Краткое описание игры:
Данная игра была написана на языке C# с помощью игрового движка Unity. 
Проект представляет из себя задание из курса по программированию в Unity Романа Сакутина - ЯЮНИОР.
<br/>
<br/>
Геймплей проекта представляет из себя следующее:
<br/>
1. Игрок управляет шариком, с помощью свайпов на экране;
2. Шарик катится слева направо и может прыгать;
3. По пути процедурно генерируется уровень с препятствиями;
4. На уровне также есть линии из монеток, которые тоже генерируются;
5. Игроку нужно собирать монетки, избегая препятствий;
6. Количество подобранных монеток отображается в верхней части экрана в виде очков игрока;
7. Игра заканчивается, если игрок вылетает с уровня или застревает в препятствии.
<br/>
В результате данный проект можно назвать полностью играбельным конечным продуктом. Игру можно развивать добавляя новый контент.
